Installation

The program will be supplied by Tony. It requires the Java Run Time Environment (JRE) to be installed on your machine

First usage

The first time you start the program use the MPCORB button to select your local copy of MPCORB.DAT.


Buttons and Controls

  • Refresh: Refresh all data.
  • (Observatory selection): Select G40 or W88.
  • Ephemeris: The MPC Ephemeris page will be displayed for those the selected non-NEOCP objects.
  • NEOCPs: The MPC Ephemeris page will be displayed for NEOCP/PCCP objects.
  • +24hrs: Show data for the following night.

    • I use robotic telescopes to capture images of Near Earth Asteroids and newly discovered comets. I measure their positions and magnitudes, and report this information to the Minor Planet Centre (MPC).

    I have devloped some small software tools to help me. Some of them are also used by my friends in the Slooh.com A-Team (a group of like-minded asteroid and comet chasers using the Slooh.com robotic telescopes.

    I am placing these tools on Github as a convenience to the A-Team members and in case they may be of any use to anyone else.

    ADES Comet

    This takes separate positional (astrometric) and magnitude (photometric) observations and combines them into single ADES observations containing both the astrometric and photometric information, ready to send to the MPC.

    ADES COMET

    ADES to Obs80

    This takes ADES data, as output from Astrometrica and converts it to the old Obs80 format

    ADES to Obs80

    NEOCP Data extract

    This downloads data from the MPC detailed records of NEOCP objects and extracts the observations in obs80 format.
